    Warhorse

    The main Warhorse class, containing all actions available to automate tasks and builds.

    Constructor

    new Warhorse(moduleDirectory, workingDirectory, options)

    The Warhorse constructor takes an options object. In these options are contained the user requested configuration defaults for the project. If there are no options - then Warhorse's own default configuration settings will be used instead.

    Parameters:
    Name Type Description
    moduleDirectory string

    Directory where this file's module is.
    workingDirectory string

    Directory where this code was originally invoked from.
    options Object

    Configuration options to override Warhorse's own defaults.
